# Repository Instructions

**This repository is PUBLIC and published to the open internet at https://tools.esub.com.**

Treat every file committed here as world-readable, permanent, and indexable by search engines.

## Hard rules

1. **No source code.** This repo hosts only self-contained, presentational HTML pages (plus their own inline/local CSS, JS, and static assets). Do not commit application source, libraries, backend logic, build tooling, or code intended for internal systems.

2. **No internal content.** Do not commit anything internal, proprietary, or confidential, including but not limited to:
   - Secrets, credentials, API keys, tokens, passwords, connection strings.
   - Internal URLs, hostnames, IPs, infrastructure or architecture details.
   - Customer, employee, or other personal/PII data.
   - Internal documents, screenshots, exports, or business data not cleared for public release.

3. **Public-only content.** Only publish material that is already approved for public disclosure. If there is any doubt about whether content is safe to make public, do not commit it — confirm first.

## What belongs here

Standalone HTML tools/pages, each in its own top-level directory (`<tool-name>/index.html`), served as static files. See `CLAUDE.md` for URL structure and deployment details.
